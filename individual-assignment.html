Introduction

The aim of this assignment is to give an example of a good and a bad user interface design.
As it is hard to find a perfect interface or a completely bad interface, I will centre my attention on specific aspects of the interfaces.
For classifying what is a good and a bad design I will focus on the following criteria:

  • LearnabilityIf it is easy for new users to learn it, begin effective interaction and achieve maximal performance
  • EfficiencyIf, once the user has learned it, a high level of productivity is expected
  • MemorabilityIf it is easy to remember
  • Error freeIf it has a low error rate
  • SatisfactionIf it is pleasant to use
